# Tampa superior do teclado solta

A tampa (ou tampo) superior do seu teclado está solta ou com folga?

:::tip Vídeo
Veja um exemplo do problema neste [vídeo](https://youtube.com/shorts/haTmtxk9azo).
:::

## Causas

A tampa superior do teclado é sustentada por pilares que são fixados com parafusos na parte interna da estrutura. As principais causas para a tampa ficar solta são:

- **Parafusos afrouxados** - Os parafusos que fixam os pilares podem ter se soltado com o tempo ou uso
- **Vibração durante transporte** - Movimentos e impactos podem causar o afrouxamento dos parafusos

## Como diagnosticar

Antes de abrir o teclado, verifique:

- A tampa apresenta movimento ou folga quando você pressiona levemente?
- A tampa está visivelmente desalinhada ou solta?

Se a resposta for sim para qualquer uma dessas perguntas, provavelmente os parafusos dos pilares precisam ser apertados.

## Solução

Para resolver o problema, você precisará abrir o teclado e apertar os parafusos que fixam os pilares da tampa. Siga os passos abaixo:

:::warning Atenção
Certifique-se de desligar completamente o teclado antes de iniciar qualquer procedimento de manutenção.
:::

1. **Desligue o teclado completamente**
   - Desconecte o cabo USB (versão Padrão)
   - Desligue as chaves de energia (versão Wireless)

2. **Desconecte as metades** (apenas versão Padrão)
   - Desconecte o cabo USB que conecta as duas metades do teclado

3. **Abra a estrutura do teclado**
   - Vire o teclado de cabeça para baixo
   - Remova os **5 parafusos** localizados na parte inferior do teclado
   - Remova cuidadosamente a parte inferior para expor o interior

4. **Localize os parafusos dos pilares**
   - Identifique os **2 parafusos internos** que fixam os pilares que sustentam a tampa superior
   - Esses parafusos estão localizados na estrutura interna, próximos aos pilares

5. **Aperte os parafusos**
   - Use uma chave philips (o teclado acompanha uma)
   - Aperte os parafusos com cuidado, sem forçar excessivamente para não danificar as roscas
   - Verifique se ambos os parafusos estão bem apertados

6. **Teste e feche o teclado**
   - Antes de fechar, verifique se a tampa está firme pressionando levemente
   - Se estiver resolvido, monte novamente a parte inferior
   - Reinstale os 5 parafusos da parte inferior
   - Reconecte as metades (versão Padrão) e ligue o teclado
